CEF Voice Example

Record audio → transcribe via AssemblyAI → detect signals via Gemini → store in a CEF vault cubby

Before this works

  1. Get a Cere wallet at wallet.dev.cere.io and fund it with devnet tokens
  2. Create an agent service in ROB (rob.compute.dev.ddcdragon.com) using that wallet — copy your agentServicePubkey
  3. Fill in .env with your pubkey and agent ID, then cd agent && npm install && npm run build && npm run prepare-manifest
  4. Upload manifest-publish.json to ROB → publish → set your AssemblyAI + Gemini keys in the agent settings UI
  5. Go to the marketplace (agent-marketplace.compute.dev.ddcdragon.com), search for your agent, connect it to your wallet — disconnect any old version first

Vault URL is resolved automatically from VITE_WALLET_ENV. No JSON signer needed — ROB signs through your browser wallet.

1. Connect

Sign in with your Cere wallet. Your vault is claimed automatically.

2. Record

Audio is captured in 5-second chunks. Each chunk is uploaded to AssemblyAI before being sent to the agent — the CEF sandbox cannot send binary HTTP bodies, so the upload happens here in the browser.
